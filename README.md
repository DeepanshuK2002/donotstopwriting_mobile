# Do Not Stop Writing

> Defeat writer's block and enter the flow state instantly. If you stop typing, your progress is deleted.

 [**donotstopwriting.com**](https://donotstopwriting.com/) | [**Live Demo & Instant Check**](https://donotstopwriting.pages.dev/)

---

## What is Do Not Stop Writing?

**Do Not Stop Writing** is a minimalist, premium online writing tool engineered for one core purpose: to help writers break through screen paralysis, conquer self-censorship, and build unshakeable output momentum. 

Commonly described as a **dangerous writing app**, it introduces a high-stakes, gamified constraint to your writing sessions. If you pause typing for more than 3 seconds (or 2 seconds in hardcore mode), the screen blurs, a warning glow intensifies, and all your progress is wiped. This forces you to prioritize raw creative output over premature editing.

---

## Features

- **Goal-Oriented Sessions:**
  - **Time Goals:** Write for 3, 5, 10, 15, 30, or 60 minutes.
  - **Word Count Goals:** Write 250, 500, 1000, or 2000 words.
  - **Custom Goals:** Set your own custom minutes or word limits.
- **Hardcore Blind Writing Mode:** Permanent text blur that only shows the word you are currently typing. Once you press space, it fades away, preventing you from looking back and self-editing.
- **Wiped Progress Prevention:** Copying, cutting, or text-selection is completely disabled while writing to keep the stakes high.
- **Multi-Format Downloads:** Once you successfully complete your goal, copy protection is lifted and you can download your draft in 14+ formats (docx, pdf, txt, odt, rtf, pages, xml, and more).
- **Clean Minimal UI:** Fully responsive design with automatic OS-themed scrollbars and cursor styles, alongside dark mode support.

## Local Development

All commands are run from the `dir/` directory:

```sh
# Install dependencies
npm install

# Start local dev server
npm run dev

# Build for production
npm run build

# Preview build locally
npm run preview
```
